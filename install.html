      Browser Support

      Direct card installation uses the File System Access API. USB installation uses Web Serial. Chromium browsers provide both on desktop; ZIP and JSON downloads work as the portable fallback.

      Field Manual

      1. Add or reorder up to eight locations.
      2. Select the microSD root or connect the Pip-Boy by USB.
      3. Choose Install App + Data for first use, then Fetch Data Only for routine updates.
      4. Reboot the Pip-Boy and open INV > APPS > Weather.

      Weather comes from Open-Meteo. Space conditions come from NOAA SWPC. No API key is required.
